# Quick Start
1. Clone this repo
2. Install Node v19.x and npm
3. Run `npm install`
4. Create a file named `.env` with the following values (google it)
    ```
    DISCORD_TOKEN="<DISCORD_BOT_TOKEN>"
    HF_TOKEN="<HUGGING_FACE_TOKEN>"
    ```
5. Run `npm run start:register` to register commands
6. Profit
